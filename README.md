# 📝 Modern To-Do List

A sleek and interactive **Modern To-Do List** web application built using HTML, CSS, and JavaScript. This project helps users manage daily tasks efficiently with a minimalist design and intuitive interface.

---

## 🔍 Features

- ➕ Add new tasks with input validation
- ✅ Mark tasks as completed
- ❌ Delete individual tasks
- 🗑️ Clear all tasks
- 💾 Local storage support to persist tasks
- 📱 Fully responsive design for all screen sizes

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript (ES6)**
